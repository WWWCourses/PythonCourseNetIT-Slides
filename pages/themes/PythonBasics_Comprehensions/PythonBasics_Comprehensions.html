Comprehensions

List Comprehensions

Why

A common programming task involves generating a new list based on the values of an existing list (or another iterable), i.e. mapping the elements of one list into another.
Let's say you want to create a list of squares of the numbers numbers in a list
We can use for loop (not Pythonic): 

                numbers = [1,3,2,5]
                squares = []
                for num in numbers:
                    squares.append(num ** 2)

                print(squares)

                # [1, 9, 4, 25]
but list comprehension offers more shorter way:

                numbers = [1,3,2,5]
                squares = [num**2 for num in numbers]

                print(squares)

                # [1, 9, 4, 25]

Syntax

List comprehensions in Python provide a concise way to create lists.

                new_list = [expression for item in iterable if condition]
where:
expression is the item to be returned and added to the new_list. I.e. it acts like a mapping expression.
item is the variable representing the members of the iterable.
iterable is the object you are iterating over.
if condition clause is optional, and if present it plays the role of filter.
Examples:

                l = [1,2,3,4,5]

                # Multiplying Each Element by 3
                tripled = [x * 3 for x in l]        # [3, 6, 9, 12, 15]

                # Filtering Even Numbers:
                evens = [el for el in l if el%2==0]	# [2, 4]

                # map letters to its uppercase
                letters = ['a', 'b', 'c']
                upper_letters = [l.upper() for l in letters]
                print(upper_letters)

Examples

Consider next task: generate a new list from a given list l, where each even number is replaced with string 'even' and each odd number - with 'odd'.
Hint: we would not need the if clause, because we do not want to filter the list, but instead - map a value for each list item. So, we must write an expression which return 1 if item is even, and 0 - if item is odd.

                numbers = [1,3,2,5]
                mapped = ['even' if num%2==0 else 'odd' for num in numbers]
                print(mapped)

                ['odd', 'odd', 'even', 'odd']

Dictionary Comprehensions

Dictionary comprehensions in Python are similar to list comprehensions but are used to create dictionaries.
A dictionary comprehension consists of an expression pair (key: value) followed by a for statement inside curly braces {}.

                new_dict = {key: value for item in iterable if condition}
where:
key is the expression that defines the key in each key-value pair.
value is the expression that defines the value in each key-value pair.
item, iterable and condition are the same like in list comprehensions.
Example: 

                eng_bul = {
                    'apple':'ябълка',
                    'orange':'портокал',
                    'banana':'банан'
                }

                # swapping keys and values
                bul_eng = {v:k for k,v in eng_bul.items()}
                print(bul_eng)

                # {'ябълка': 'apple', 'портокал': 'orange', 'банан': 'banana'}

Examples

Create new dictionary from student_scores, containing only the items which has values > 4.0

                student_scores = {
                    'ivan':4.5,
                    'maria':5.0,
                    'asen':3.5
                }

                best_students = {k:v for k,v in student_scores.items() if v>4}

                print(best_students)

                # {'ivan': 4.5, 'maria': 5.0}
Creating a dictionary of even numbers and their squares:

                even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
                print(even_squares)

                # {0: 0, 2: 4, 4: 16, 6: 36, 8: 64}
Creating a Dictionary from Two Lists

                keys = ['a', 'b', 'c']
                values = [1, 2, 3]
                dictionary = {keys[i]: values[i] for i in range(len(keys))}
                print(dictionary)

                # {'a': 1, 'b': 2, 'c': 3}
