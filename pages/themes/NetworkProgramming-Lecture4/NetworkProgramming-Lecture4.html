HTTP Client with requests lib

Getting Started

Requests allows you to send HTTP/1.1 requests extremely easily. There’s no need to manually add query strings to your URLs, or to form-encode your POST data. Keep-alive and HTTP connection pooling are 100% automatic, thanks to urllib3.
Reference: Requests lib official site
Make sure you have latest version of requests installed

				$ pip show requests
					Name: requests
					Version: 2.27.1
					Summary: Python HTTP for Humans.
					Home-page: https://requests.readthedocs.io
					Author: Kenneth Reitz
Else:

				pip install requests
			

				pip install --upgrade requests

Syntax:

The requests module allows us to send HTTP requests using Python.

				requests.methodname(params)
Usage:

				import requests

				response = requests.get('https://softwareacademy.bg/')

				print(response.content)

Response Object

Example: Send GET Request


			# base url
			url = 'https://httpbin.org/get'

			# prepare headers
			headers = {
				'User-Agent':'Mozilla/5.0 (X11; Linux x86_64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/98.0.4758.102 Safari/537.36'
			}

			# prepare query params
			query = {
				'param1':'value1',
				'param2':'value2',
			}

			# send request
			response = requests.get(url, headers=headers, params=query)

			# print response
			if response.ok:
				print(response.text)
				print(response.headers)

Example: access Request object from Response


			import requests

			# requests.get() send the request and returns a Response Object:
			response = requests.get('https://softwareacademy.bg/')

			### We can access the request object from the response as:
			print(response.request)

			# Let's see the deafult headers sent:
			print(response.request.headers)

Simple HTTP Server with http.server

Simple HTTP Server with http.server

Create and run the server

Python's built-in module http.server defines classes for implementing HTTP servers

				from http.server import HTTPServer,BaseHTTPRequestHandler

				# make server address
				ip = '127.0.0.1'
				port = 8080
				server_address = (ip, port)

				# instantiate the server
				httpd = HTTPServer(server_address, BaseHTTPRequestHandler)

				# start server to listen for client's connections
				print(f'HTTP Server is listening on {ip}:{port}')
				httpd.serve_forever()
server_address- The address on which the server is listening. For internet protocols this is a tuple containing a string for server IP or name, and an integer port number
Note that the code above will only run the server, but it will not be able to answer any requests.

Create requests handler class

To handle any requests, the serve must subclass the BaseHTTPRequestHandler and respective methods

				class RequestHandler(BaseHTTPRequestHandler):
					def do_GET(self):
						# setup header
						self.send_response(200)
						self.send_header("Content-type", "text/html")
						self.end_headers()

						# setup body and send the response
						response_body = '
Server is working!
'.encode('utf-8')
						self.wfile.write(response_body)
The handler will parse the request and the headers, then call a method specific to the request type, like do_GET, do_POST, do_PUT, do_DELETE and so on.
Note, that we must encode the response body to bytes before sending the response.
