Overview

pandas is an open source library providing high-performance, easy-to-use data structures and data analysis tools for the Python programming language
pandas is great for data analysis and modeling
The two primary data structures in pandas are Series and DataFrame
Pandas home page

Install in Python's venv

You should install pandas in activated Python environment

			# create Python environment (if you do not have one):
			# note that executing next command will create virtual environment (named '.venv') in current working directory:
			$ python3 -m venv .venv

			# activate Python (.venv) environment in Bash:
			$ source .venv/bin/activate

			# activate Python (.venv) environment in Command Prompt:
			$ .venv\Scripts\activate.bat

			# install pandas
			(.venv)$ pip install pandas

Use Pandas

You can start using pandas in any Python file

				import pandas as pd

				df_users = pd.DataFrame({
					"name":['Maria', 'Petyr', 'Ivan'],
					"age":[23, 21, 34],
					"gender":['female', 'male', 'male']
				})

Pandas Data Structures Overview

Pandas Data Structures Overview

The two primary data structures in pandas are Series and DataFrame
A DataFrame is a collection of Series objects
A Pandas Series is a one-dimensional array of indexed data, based on the NumPy ndarray. Pandas Series object wraps both a sequence of values and a sequence of indices
a DataFrame is an analogue of a two-dimensional array or table with flexible row and column indices.
You can think of DataFrame as an Excel table, and of Series - as a column in an Excel table

DataFrame as a collection of Series

A multi-column DataFrame can be thought as a dictionary of Series objects, where the dictionary keys represent the columns names. Note, that the Series objects must be aligned, i.e. to have the same index.

			prices_ds = pd.Series([1.5, 2, 2.5, 3],
			index=["apples", "oranges", "bananas", "strawberries"])

			suppliers_ds = pd.Series(["supplier1", "supplier2", "supplier4", "supplier3"],
			index=["apples", "oranges", "bananas", "strawberries"])

			fruits_df = pd.DataFrame({
				"prices": prices_ds,
				"suppliers": suppliers_ds
			})
			print(fruits_df)
			#              prices  suppliers
			#apples           1.5  supplier1
			#oranges          2.0  supplier2
			#bananas          2.5  supplier4
			#strawberries     3.0  supplier3
