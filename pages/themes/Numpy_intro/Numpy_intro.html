NumPy Intro

NumPy overview

NumPy is an essential tool in the Python ecosystem for anyone working with numerical data, scientific computing, or machine learning, providing a robust foundation for data manipulation and computation.
NumPy provides the array object, which is a powerful N-dimensional array capable of holding data of the same type.
It includes functions for performing element-wise operations, linear algebra, statistical computations, and much more.
Written in C, it offers high performance for mathematical operations, making it suitable for large-scale data processing.
Supports vectorization, which reduces the need for explicit loops, enhancing computational efficiency.
NumPy serves as the backbone for a broad array of Python libraries in the scientific computing and data analysis ecosystem.

Key modules and libraries integrated with numpy

SciPy - scientific and technical computing
Pandas - data analysis and data science tool
Matplotlib - comprehensive library for creating static, animated, and interactive visualizations in Python.
Statsmodels - library for statistical modeling and econometrics
Scikit-learn - a machine learning library
TensorFlow and PyTorch - libraries for deep learning and neural networks

N-Dimensional Arrays

The main object in numpy is the homogeneous N-Dimensional Array, called ndarray Object.
It is a sequence of elements (usually numbers), all of the same type, indexed by a tuple of positive integers for each dimension.
In numpy, dimensions are called axes
Number of axes defines the rank of the array

Example


			import numpy as np

			# create one-dimensional array
			arr1d = np.array([1, 2, 3])
			# create two-dimensional array
			arr2d = np.array([[1, 2, 3], [4, 5, 6]])

			print(f'arr1d: \n{arr1d}\n')
			print(f'arr2d: \n{arr2d}\n')

			print(f'arr1d.shape: {arr1d.shape}') # (rank 1, axes 0)
			print(f'arr2d.shape: {arr2d.shape}') # (rank 2, axes 0 and 1)
		

			arr1d:
			[1 2 3]

			arr2d:
			[[1 2 3]
			[4 5 6]]

			arr1d.shape: (3,)
			arr2d.shape: (2, 3)

NumPy arrays vs Python's lists

Implementation - numpy arrays are homogeneous. This allows NumPy to optimize operations and memory usage effectively.

Array with Mixed Data Types?

If you try to create a NumPy array with elements of different types, NumPy will upcast elements to a common data type that can accommodate all the values, usually converting everything to a type that can hold all the given data.

				import numpy as np

				mixed_array = np.array([1, 2.5, '3'])
				print(mixed_array.dtype)  # Output: <U32 (Unicode string)

NumPy Data Types

NumPy Data Types

NumPy provides a variety of data types, referred to as dtypes, which are more efficient than Python’s native data types.
These dtypes are crucial for memory efficiency and speed in numerical computations.
There are 5 basic numerical types representing booleans (bool), integers (int), unsigned integers (uint) floating point (float) and complex.
Do not forget, that Numpy Arrays are homogeneous! I.e. can not store data from different data types.
