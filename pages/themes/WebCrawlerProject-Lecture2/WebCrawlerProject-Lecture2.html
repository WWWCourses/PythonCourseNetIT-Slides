request lib usage:

The requests module allows us to send HTTP requests using Python.

				requests.methodname(params)
Where:
params – (optional) Dictionary, list of tuples or bytes to send in the query string for the Request.
data – (optional) Dictionary, list of tuples, bytes, or file-like object to send in the body of the Request.
json – (optional) A JSON serializable Python object to send in the body of the Request.

Using BeautifulSoup

Using BeautifulSoup

Overview

Beautiful soup is one of the most widely-used Python libraries for web scraping.
It works with your favorite parser to provide idiomatic ways of navigating, searching, and modifying the parse tree.
Installation:

				pip install beautifulsoup4
Official Documentation

Beautiful soup hands on

Let's extract links for each publication in https://bnr.bg/hristobotev/radioteatre/list
The inspection of the site shows that the links we needed are in div with id="module_1_1"

Create and navigate in the soup

First we need to create a BeautifulSoup object, which will represents the document as a nested data structure.
Then using variety of properties and methods we can navigate into the tree in order to get the meaningful information
Reference: Navigating the tree

			from bs4 import BeautifulSoup as bs

			# create BeautifulSoup object, which represents the document as a nested data structure:
			soup = bs(html, 'html.parser')

			# get HTML element:
			print(soup.title)

			# # get HTML element's content as string:
			print(soup.title.string)
